## using

1. 설문 자료 엑셀 다운로드
2. `data.csv`로 이름변경
3. github codespace로 저장소열기
4. `data.csv` project root로 업로드
5. 우분투 image-to-ascii 필요 모듈설치
  ```
  # Ubuntu
  $ sudo apt-get install streamer
  ```
4. yarn install & npm install
5. yarn start && (재추첨) yarn re
